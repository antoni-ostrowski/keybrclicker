# Development

Build: `just build`.
Run the output: "./bin/keybrclicker" or "just start".
Use simple/functional patterns. Keep everything in `main.swift` in the src dir (it contains all the source code).
Read BIBLE.md for app description for context.
When changing some user facing UX or config or anything that user needs to know, always update the readme about the new way of doing stuff so its always up to date.
